# Medical 관련 프로젝트

의료 산업 관련 프로젝트로, 오처방 탐지 모형 개발과 안구 영상 탐지 프로젝트가 포함.

## Data
Pannel Data와 Image DAta가 포함되어 있음.

- **matplotlib**    ≥       `3.0.2`
- **numpy**           ≥      `1.16.0`
- **opencv-python**    ≥     `3.4.3.18`
- **pandas**        ≥        `0.23.4`
- **polarTransform**    ≥    `2.0.0`
- **scikit-learn** ≥         `0.20.2`
- **scipy**      ≥           `1.2.0`
- **sklearn**      ≥         `0.0`
- **tensorboard**   ≥        `1.12.2`
- **tensorflow**  ≥         `1.12.0`
- **tflearn**      ≥         `0.3.2`
- **tqdm**         ≥         `4.31.1`
